Comment utiliser cette application :

  1. Entrez un titre pour votre tirage au sort dans le champ "Titre".
  2. Dans la zone "Utilisateurs", entrez un nom par ligne pour chaque participant.
  3. Dans la zone "Catégories", entrez une catégorie par ligne. Si vous laissez cette zone vide, des catégories seront automatiquement créées en fonction du nombre d'utilisateurs.
  4. Cliquez sur le bouton "Tirer au sort" pour effectuer le tirage.
  5. Les résultats s'afficheront dans un tableau en dessous.
  6. Utilisez le bouton "Imprimer" pour imprimer ou sauvegarder les résultats.

Note : Chaque utilisateur sera assigné à une catégorie de manière aléatoire. Si le nombre d'utilisateurs dépasse le nombre de catégories, les catégories seront réutilisées de manière cyclique.
